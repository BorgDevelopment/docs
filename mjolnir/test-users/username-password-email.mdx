Use this when login sends a one-time verification code to the user's email address that must be entered to complete sign-in.

## What to provide

```text
Username: testuser@example.com
Password: YourTestPassword123
Email inbox access: [explain how Mjolnir can retrieve the verification code]
```

## Providing inbox access

Mjolnir needs a way to retrieve the code from the inbox automatically. Options:

<Tabs>
  <Tab title="Test email service (recommended)">
    Use a service like [Mailosaur](https://mailosaur.com) or [Mailtrap](https://mailtrap.io) that provides programmable inboxes with an API. Include the API key and inbox ID in the instructions.

    This is the most reliable option. It avoids IMAP timeouts and gives Mjolnir a clean, fast way to retrieve codes.
  </Tab>
  <Tab title="Email forwarding webhook">
    Set up a forwarding rule that POSTs incoming emails (or just the code) to a webhook URL. Tools like [Mailhook](https://mailhook.io) or a simple webhook receiver work well.
  </Tab>
  <Tab title="IMAP credentials">
    Provide IMAP access to the inbox:

    ```text
    IMAP host: imap.example.com
    IMAP port: 993
    IMAP username: testuser@example.com
    IMAP password: imappassword
    ```
  </Tab>
</Tabs>
